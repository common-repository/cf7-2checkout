Twochout payment gateway integrated with contact form 7

== Description ==

"Contact Form 7 2Checkout" is a WordPress plugin which allows the site owners to accept credit cards through 2Checkout Payment Gateway integrated with contact form 7 plugin. It takes as little as 5 minutes to set up and is by far the quickest way to integrate 2Checkout Payment gateway. This plugin allow you to accept transactions from all major credit cards including Visa, MasterCard, American Express, and Discover. Direct payment API is integrated with CF7 customer will not be directed to 2Checkout site for payments, rather credit cards will be processed on your site. 
The plugins requires SSL to be active on your site otherwise payments will not be processed.
